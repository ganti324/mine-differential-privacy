# Differential Privacy Notebook Gallery

This directory contains Jupyter notebooks demonstrating the usage of the Differential Privacy libraries in various languages.

## Notebooks

- **[BoundedSum_Java.ipynb](BoundedSum_Java.ipynb)**: Demonstrates the `BoundedSum` algorithm using the Java library.
- **[ApproxBounds_CPP.ipynb](ApproxBounds_CPP.ipynb)**: Demonstrates the `ApproxBounds` algorithm using the C++ library.
- **[DP_Mechanisms_Python.ipynb](DP_Mechanisms_Python.ipynb)**: Demonstrates basic DP mechanisms (`LaplaceNoise`, `GaussianNoise`) using the Python library.

## Prerequisites

To run these notebooks, you will need:

- **JupyterLab** or **Jupyter Notebook**
- **Kernels**:
  - **Java**: [IJava](https://github.com/ SpencerPark/IJava) kernel.
  - **C++**: [xeus-cling](https://github.com/jupyter-xeus/xeus-cling) kernel.
  - **Python**: Standard Python 3 kernel with `differential-privacy` library installed.

## Running the Notebooks

1. Install the necessary kernels.
2. Start Jupyter:
   ```bash
   jupyter lab
   ```
3. Open the desired notebook and run the cells.
